nsite gateway

nostr の静的サイト（nsite）を <npub>.nsite.compile-error.net で配信するゲートウェイです。

サイトを見る

あなたの npub をサブドメインに付けてアクセスします:

https://<あなたのnpub>.nsite.compile-error.net/

例: npub1937…auhaj6.nsite.compile-error.net

サイトを公開する

NIP-5A の nsite を発行するだけです。① blossom サーバへファイルをアップロードし、② kind 15128 のマニフェストを、このゲートウェイが見ているリレーへ publish します。

このゲートウェイが参照するリレー:

  • wss://nostr.compile-error.net
  • wss://relay.damus.io
  • wss://nos.lol

nostr-nsite-server の update コマンドを使う場合:

nostr-nsite-server update \
  -sec <nsec | bunker://...> \
  -server https://blossom.compile-error.net \
  -relay wss://nostr.compile-error.net \
  ./your-site-dir

マニフェストは replaceable なので、再実行すれば上書き更新されます。反映は数十秒以内です。
